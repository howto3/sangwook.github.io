Confusion matrix 를 읽는 법

2013-06-04

Confusion Matrix 는 아래와 같이 나온다.

=== Confusion Matrix ===

   a b

읽는 법은

  • row 는 actual class 고 column 은 predicted class 다.
  • 즉, 실제로 b로 분류된게 a로 잘못 예측된건 73건이고. b로 올바르게 예측된건 136개 라고 읽으면 된다.
